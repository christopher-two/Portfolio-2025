# Historial de Versiones - App Spot

Manténgase al día con las últimas mejoras y funciones de Spot.

## Versión 1.2.0 (Próximamente)
- **Nuevos Filtros de Mapa**: Encuentra Spots por categoría (Tesoros, Recuerdos, Mensajes).
- **Modo Offline**: Guarda la ubicación de tus Spots para verlos después sin conexión.
- **Optimización de Batería**: Mejoras en el uso del GPS en segundo plano.

## Versión 1.1.0
- **Cambio a Cápsulas Visuales**: Ahora puedes anclar fotos además de mensajes de texto.
- **Puntos Brillantes**: Rediseño del mapa para mostrar Spots con efectos de brillo dinámicos.
- **Mejora en Precisión GPS**: Reducción del radio de bloqueo a 15 metros para mayor exclusividad.

## Versión 1.0.0
- **Lanzamiento Inicial**: Sistema básico de geolocalización y "anclaje" de mensajes.
- **Integración con Supabase**: Autenticación y base de datos en tiempo real.
- **Estética Neobrutalista**: Interfaz limpia en modo oscuro con acentos neón.
